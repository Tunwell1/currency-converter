# 💱 Currency Converter

<details>
  <summary>🇷🇺 Описание на русском (Russian Version)</summary>

  Современное веб-приложение для конвертации валют в реальном времени, построенное на **React**, **TypeScript** и **Vite**.

  ### 🚀 Особенности
  * **Real-time Conversion:** Автоматический пересчет при изменении суммы или валюты.
  * **Debounced Requests:** Оптимизация API-запросов (задержка 500мс) для предотвращения лишней нагрузки.
  * **Bi-directional:** Удобное переключение между «из» и «в» валюту с мгновенным обновлением значений.
  * **Visual Aids:** Отображение флагов стран для каждой выбранной валюты.
  * **Smart Formatting:** Динамическое отображение знаков после запятой (без лишних нулей).

  ### 🛠 Стек технологий
  * **Framework:** React
  * **Language:** TypeScript
  * **Build Tool:** Vite
</details>

<details>
  <summary>🇺🇸 English Description (Английская версия)</summary>

  A modern, real-time currency conversion web application built with **React**, **TypeScript**, and **Vite**.

  ### 🚀 Features
  * **Real-time Conversion:** Instant recalculation as you update amounts or switch currencies.
  * **Debounced Requests:** API optimization using a 500ms debounce to prevent unnecessary network load.
  * **Bi-directional Sync:** Seamlessly switch between "From" and "To" fields with immediate value updates.
  * **Visual Aids:** Integrated country flags for every selectable currency for better UX.
  * **Smart Formatting:** Dynamic decimal handling that strips trailing zeros for a clean UI.

  ### 🛠 Tech Stack
  * **Framework:** React
  * **Language:** TypeScript
  * **Build Tool:** Vite
</details>
